# QuizMaster – Java Quiz Platform

QuizMaster is a Java-based quiz application that allows users to register, log in, take quizzes, and view reports.

## Features
- User registration & login
- Start a quiz
- Score calculation
- Reports & history
- Modular OOP-based design

## Technologies Used
- Java (OOP)
- Collections Framework
- UML & System Design

## How to Run
1. Install JDK 17+
2. Compile using:
   javac Main.java
3. Run using:
   java Main
